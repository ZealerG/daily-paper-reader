## 摘要
我们提出了一种用于多智能体协调的可微优化框架。输入被分解为重叠的局部视图，每个视图由一个智能体处理，该智能体求解由神经编码器参数化的凸子问题。智能体通过交替方向乘子法(ADMM)进行协调，其间智能体之间的约束由细胞层(cellular sheaf)指定。该层指定相邻解必须在哪些方面达成一致，从而允许异构的全局共识概念。通过反向传播经过展开的优化过程，联合训练多智能体系统的所有组件。我们在迷宫路径规划、图像分类和数独上进行了评估，在这些任务中，局部视图单独不足的智能体学习协调以产生正确的全局输出。在MNIST上，局部视图分解相比标准CNN提高了对分布变化的鲁棒性。在数独上，由优化推导出的结构产生了比参数匹配的MPNN基线显著更高的求解率。最后，ADMM结构暴露了不同的原始、共识和对偶状态变量，使得协调动态可以直接分析和干预——这是标准消息传递架构所不具备的特性。

## Abstract
We present a differentiable optimization framework for multi-agent coordination. An input is decomposed into overlapping local views, each processed by an agent that solves a convex subproblem parameterized by a neural encoder. Agents coordinate through the Alternating Direction Method of Multipliers (ADMM) with inter-agent constraints specified by a cellular sheaf. The sheaf specifies which aspects of neighboring solutions must agree, allowing for heterogeneous notions of global consensus. Backpropagating through the unrolled optimization jointly trains all components of the multi-agent system. We evaluate on maze pathfinding, image classification, and Sudoku, where agents with individually insufficient local views learn to coordinate to produce correct global outputs. On MNIST, the local-view decomposition yields improved robustness to distribution shifts relative to a standard CNN. On Sudoku, the optimization-derived structure yields markedly higher solve rates than parameter-matched MPNN baselines. Finally, the ADMM structure exposes distinct primal, consensus, and dual state variables, opening the coordination dynamics to direct analysis and intervention -- a property unavailable in standard message-passing architectures.

## 2. 论文提出的方法论：核心思想、关键技术细节、算法流程

### 核心思想
- 将输入分解为**重叠的局部视图**，每个视图对应一个智能体。
- 每个智能体维护三组变量：**原始变量 x**（局部提议）、**共识变量 z**（全局一致状态）、**对偶变量 u**（累积不一致）。
- 智能体之间通过**细胞层（sheaf）** 定义的**线性限制映射**（restriction maps）进行通信：只要求投影到共享边空间的部分一致，而非全状态一致。
- 使用**ADMM迭代**实现协调，迭代过程可微，通过**算法展开（algorithm unrolling）** 反向传播训练所有组件（编码器、解码器、限制映射、惩罚参数等）。

### 关键技术细节
- **输入分解**：将输入图/图像划分为N个重叠窗口，每个窗口对应一个智能体；智能体之间根据空间关系（如网格4-连通或8-连通）构成图。
- **局部目标函数 f_i**：由共享编码器 Enc_θ 输出参数（如二次型 Q_i, q_i 或加上 ℓ1 正则化），求解凸子问题（x-更新）：
  - 二次型：`x_i^(k+1) = (Q_i + ρI)^{-1}(ρ(z_i^k - u_i^k) - q_i)`
  - 对角+ℓ1+箱约束：封闭形式包含软阈值和裁剪。
- **细胞层限制映射 F_{i→e}**：每个方向学习一个基础映射（共享），并支持低秩（LoRA）调制以根据局部内容调整。
- **z-更新（共识步骤）**：投影到约束集 C = {z | Fz = 0}（或软约束 γ/2 ||Fz||²），通过**共轭梯度法**或梯度下降（扩散步）逼近投影。
- **u-更新**：`u^(k+1) = u^k + x^(k+1) - z^(k+1)`
- **解码**：解码器 Dec_ψ 将最终 x_i^K 和局部视图映射为局部预测，再聚合为全局输出。
- **训练**：端到端，反向传播通过K次ADMM迭代和T次内循环（共轭梯度）。损失函数为任务损失（如交叉熵），并可采用**后几轮平均损失**（loss window）训练。

### 算法流程（文字描述）
```
输入：图像/图 D，智能体图 G，超参数 K, T, ρ
编码：为每个智能体 i，用 encoder(d_i) 输出局部目标参数及限制映射调制
初始化：z=0, u=0
循环 k=1..K:
  并行 x-更新：每个智能体求解 prox_{f_i}(z_i - u_i)
  z-更新：执行 T 步共轭梯度/扩散，使 z 接近 ker(F)（或带软惩罚）
  u-更新：u += x - z
解码：用 decoder(d_i, x_i^K) 输出局部预测，聚合为全局预测
```

## 6. 论文的主要结论与发现

- **Sheaf-ADMM能有效协调局部智能体**，在迷宫（99.9%求解率）、MNIST（98.5%准确率）、数独（92.6%求解率）上均达到或超越最强MPNN基线。
- **在数独上，Sheaf-ADMM的求解率（92.6%）远超参数匹配的MPNN（10.7%）**，且参数量相近（1.12M vs 1.15M）。
- **在MNIST分布偏移下，Sheaf-ADMM的鲁棒性显著优于CNN**（如填充16像素时86.3% vs 11.4%）。
- **细胞层结构至关重要**：使用固定恒等映射或仅共享映射时性能大幅下降；LoRA调制在低维度时提升明显，而通过增加d_v/d_e容量，静态共享映射也可接近LoRA性能。
- **共轭梯度（CG）优于梯度下降+Nesterov**，尤其在MNIST和数独上，而梯度下降在数独上完全失败（0%）。
- **ADMM迭代次数K影响显著**：从K=1到K=20-30性能上升，超过后可能下降（梯度退化）。
- **ADMM结构提供可解释性**：可以观察原始/残差/对偶变量，分析协调动态（如图4显示后期对偶残差集中在路径分支点）。

## 7. 优点

1. **方法创新性**：将ADMM与细胞层结合，实现可微的、结构化的多智能体协调，区别于纯学习型消息传递。
2. **可解释性**：通过显式维护x, z, u三个状态，能够分析协调过程，诊断瓶颈（如哪个区域的残差未收敛）。
3. **归纳偏置强**：优化推导的更新规则（近端映射、Laplacian扩散）比任意学习映射更稳定，利于外推和鲁棒性。
4. **分布外泛化**：局部视图和共享权重使模型可自然处理大小变化（迷宫外推）和输入噪声（MNIST鲁棒性）。
5. **灵活性**：支持多种局部目标（二次、ℓ1、QP）和多种共识形式（硬约束、软惩罚），适用于不同任务需求。
6. **实验充分**：在三个典型任务上验证，消融系统，对比公平。

## 8. 不足与局限

1. **计算成本**：每轮ADMM需要求解局部子问题（可能是线性系统）和共识步骤（共轭梯度迭代），相比简单消息传递计算开销更大。论文未报告训练/推理时间。
2. **与问题结构的依赖**：假设任务可分解为重叠局部子问题且可局部一致性约束。若全局信息无法通过局部投影传递（如需要非局部统计），则框架可能失效。
3. **长距离协调限制**：当智能体图直径很大时（如迷宫尺寸远超训练时），固定K和T可能不足以传播信息，导致性能骤降（图6中4×外推仅4.5%）。
4. **限制映射学习可能退化**：若Edge Stalk维度接近Vertex Stalk维度，限制映射可能退化为简单传递，丢失压缩作用（附录表5）。
5. **非凸问题依赖**：虽然内部ADMM是凸的，但整体（编码器+解码器）非凸，收敛性缺乏保证。论文承认可以扩展到非凸但未探索。
6. **硬件资源未公开**：论文未说明具体使用的GPU、训练时长，不利于复现和比较计算成本。
