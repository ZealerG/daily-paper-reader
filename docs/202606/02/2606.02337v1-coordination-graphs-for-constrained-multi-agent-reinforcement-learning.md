## 摘要
约束多智能体强化学习面临两个相互交织的挑战：联合动作空间随智能体数量呈指数增长，以及额外的约束以奖励结构本身无法捕捉的方式耦合智能体。我们提出了面向约束多智能体强化学习的协调图（CG-CMARL），一个通过结合协调图与拉格朗日对偶性来解决这两个挑战的框架。该系统将联合问题分解为成对区域，每个区域由一组共享的Q函数服务，其中一个用于主要目标，每个约束对应一个，因此学习模型的数量与智能体数量无关。在执行时，Max-Sum消息传递在因子图上协调动作，而拉格朗日乘子控制目标与约束之间的权衡，使得单个训练模型无需重新训练即可描绘帕累托前沿。我们在温和条件下提供了收敛保证，以及一个组合误差界，它分解为可解释的独立源，每个源可追溯到特定的设计选择并可独立控制。在协作导航任务上（最多10个智能体的团队必须协调以到达目标位置，同时满足成对约束）的实验表明，我们的方法产生的帕累托前沿优于在固定奖励塑造比率下训练的既有基线，并且能够扩展到集中式方法变得棘手的大团队规模。

## Abstract
Constrained Multi-agent reinforcement learning (CMARL) faces two intertwined challenges: the joint action space grows exponentially with the number of agents, and additional requirements couple agents in ways that reward structure alone does not capture. We introduce Coordination Graphs for Constrained Multi-Agent Reinforcement Learning (CG-CMARL), a framework that addresses both challenges by combining coordination graphs with Lagrangian duality. The system decomposes the joint problem into pairwise regions, each served by a set of shared Q-functions, one for the primary objective and one for each of the constraints, so that the number of learned models is independent of the number of agents. At execution time, Max-Sum message passing coordinates actions across the factor graph, while a Lagrangian multiplier controls the objective--constraint tradeoff, allowing a single trained model to trace a Pareto front without retraining. We provide convergence guarantees under mild conditions, together with a compositional error bound that decomposes into separate interpretable sources, each traceable to a specific design choice and independently controllable. Experiments on cooperative navigation tasks (where teams of up to 10 agents must coordinate to reach target positions while satisfying pairwise constraints) show that our method produces Pareto fronts dominating established baselines trained at fixed reward-shaping ratios, while scaling to team sizes where centralized approaches become intractable.

## 2. 方法论

- **核心思想**：将全局问题分解到成对区域（pairwise regions），每个区域共享一组Q函数（一个用于主要目标，一个用于每个约束），利用协调图与Max-Sum消息传递实现去中心化动作协调，并通过拉格朗日乘子控制目标与约束的权衡。
- **关键技术细节**：
  - **两Q函数架构**：每个区域维护两个Q函数：`Qprim,R`（主目标）和`Qcost,R`（约束），两者独立训练。约束Q函数在训练时使用主Q函数贪心策略下的TD目标，确保预测的约束奖励对应于执行主策略时的预期代价，而非约束最小化策略下的代价。
  - **增强Q函数**：动作选择时，`Qaug,R = Qprim,R + λ Qcost,R`，其中λ为拉格朗日乘子。λ可在线调节，无需重新训练。
  - **Max-Sum消息传递**：在因子图上迭代交换因子到变量和变量到因子的消息，解决联合动作选择问题。采用阻尼技术（damping coefficient=0.3）以稳定环状图上的收敛。
  - **参数共享**：所有成对区域共享一个两Q头部网络，学习模型数量恒为O(1)（与智能体数N无关）。
  - **拉格朗日乘子更新**：每个智能体维护局部乘子λi，每回合结束后根据观测到的约束奖励进行投影对偶上升。行动选择时使用平均乘子¯λ。
  - **探索机制**：在Max-Sum前向Q表格中添加高斯噪声，实现协调探索。
- **算法流程**：每步构建区域观测→计算增强Q函数→阻尼Max-Sum消息传递K次→每个代理选择argmax动作→执行并收集区域转移样本→从重放缓冲采样，更新两Q头部→每回合末更新局部乘子。
- **理论保证**：在假设（A1）独立转移、（A2）奖励可分解、（A3）Slater条件下，证明了两Q函数几乎必然收敛（定理4.1），以及原对偶收敛到最优解（定理4.2）。还给出了组合误差界（定理4.4），分解为结构误差、协调误差、采样误差和函数逼近误差，各源独立可控。

## 3. 实验设计

- **场景**：使用MPE套件中的Simple Spread环境，N个智能体需要覆盖N个地标，同时避免碰撞。N ∈ {3,4,6,10}。环境满足独立转移和成对奖励结构。
- **Benchmark**：对比方法包括：
  - IQL（独立Q学习）
  - QMIX（单调值分解）
  - DCG（深度协调图，无约束处理）
  - MAPPO（多智能体PPO）
  - MAPPO-Lagrangian（多智能体PPO加拉格朗日乘子）
- **评估方式**：绘制覆盖度（覆盖地标百分比） vs 平均每步碰撞对数的帕累托前沿。所有基线通过固定碰撞惩罚混合比例（α∈{0.0,0.3,0.6}）训练，每个模型只得到一个操作点；MAPPO-Lagrangian冻结其学习到的乘子，也只得到一个点。CG-CMARL通过改变λ从单一模型绘制完整前沿。

## 6. 主要结论与发现

- CG-CMARL的帕累托前沿在所有团队规模（N=3,4,6,10）上**支配**所有基线：在相同安全水平下取得更高覆盖度，或在相同覆盖度下碰撞更少。特别是中等λ范围（0.05~1.0），前沿处于一个基线无法达到的区域。
- **单模型扫描即得帕累托前沿**：避免为每个权衡点重新训练，极大降低了应用成本。
- **可扩展性**：在N=10时仍保持优势，且方差低于MAPPO。
- **理论贡献**：给出了收敛保证和可分解的误差界，为结构设计提供了指导。

## 7. 优点

- **创新性**：首次将协调图与拉格朗日对偶无缝结合，同时解决可扩展性和约束满足。
