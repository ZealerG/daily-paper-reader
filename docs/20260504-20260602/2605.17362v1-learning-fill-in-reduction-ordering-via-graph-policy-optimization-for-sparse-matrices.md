## 摘要
大型稀疏求解器中的矩阵重排序旨在寻找一种排列，以最小化分解过程中的填充量，从而减少内存和计算。由于最小填充排序问题是NP完全的，且填充量隐含在稀疏模式中，因此采用了图论启发式方法。现有的强化学习方法要么忽略稀疏模式（从而遗漏全局填充量），要么缺乏局部精确的填充反馈。我们提出了一种图策略优化方法，从全局和局部视角建模填充量：策略网络和价值网络都使用多跳图神经网络骨干来嵌入全局填充量；策略进一步与图上的符号分解交互，以提取局部、步骤级别的填充量，并通过自适应饱和函数将所得反馈与价值网络对齐，以改善收敛性。在SuiteSparse矩阵集合上，我们的方法在填充量上平均减少29.3，在峰值内存使用上平均减少31.3，优于最先进的基线方法。

## Abstract
Matrix reordering in large sparse solvers seeks a permutation that minimizes factorization fill-in to reduce memory and computation. Because the minimum fill-in ordering problem is NP-complete and fill-in is implicit in the sparsity pattern, graph-theoretic heuristics are used. Existing reinforcement learning methods either ignore sparsity patterns--missing the global fill-in--or lack local exact fill-in feedback. We propose a graph policy optimization method, modeling fill-ins from global and local views: both the policy and value networks use a multi-hop graph neural backbone to embed global fill-in; the policy further interacts with symbolic factorization over graphs to extract local, step-level fill-ins, and the resulting feedback is aligned with the value network via an adaptive saturation function to improve convergence. On the SuiteSparse Matrix Collection, our method achieves mean reductions of 29.3 in fill-ins and 31.3 in peak memory usage over state-of-the-art baselines.

### 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

- **核心思想**：将稀疏矩阵的邻接图作为状态，节点消除作为动作，符号分解过程作为环境动力学，用负的填充数作为即时奖励，训练一个 Actor-Critic 网络来学习最优消除顺序。
- **关键技术细节**：
  - **状态表示**：当前消除图 \(G_t\) 及其节点特征（度、集体影响力 CI）。
  - **动作**：从当前图 \(V_t\) 中选择一个节点消除。
  - **奖励**：\(r_t = -|F_t|\)，其中 \(F_t\) 是当前步骤新增的填充边数。
  - **自适应饱和回报（Adaptive Saturation Return, ASR）**：\(ASR_t = \frac{|E_t| + R_t}{|E_t| - R_t}\)，其中 \(|E_t|\) 是当前边数，\(R_t\) 是从 t 步到结束的累计负奖励（即总填充数的负数）。该设计可降低方差、稳定训练。
  - **网络结构**：策略网络和价值网络均使用 MixHopConv 层（多跳聚合）提取图嵌入，后接线性层输出。策略网络输出节点消除概率分布（log softmax），价值网络输出标量状态值。
  - **损失函数**：Actor 损失 \(L_a = -\frac{1}{n}\sum_{t} \pi(v_t|G_t) \cdot A_t\)，Critic 损失 \(L_c = \frac{1}{n}\sum_{t} A_t^2\)，其中优势 \(A_t = ASR_t - \text{Value}(G_t)\)。
- **算法流程（文字说明）**：
  1. 初始化当前图 \(G_0\)，节点特征 \(X_{V_0}\)。
  2. 对每个时间步 \(t=0\) 到 \(n-1\)：
     a. GNN 前向计算策略分布 \(\pi(V_t|G_t)\) 和状态值 \(\text{Value}(G_t)\)。
     b. 从策略中采样一个节点 \(v_t\) 消除。
     c. 执行符号分解函数 `NodeElimination`，得到新图 \(G_{t+1}\)、新边集 \(F_t\) 及新节点特征。
     d. 记录策略概率、状态值、奖励 \(r_t = -|F_t|\) 和边数 \(|E_t|\)。
  3. 计算自适应饱和回报 \(ASR\) 和优势 \(A\)。
  4. 计算 Actor 和 Critic 损失，更新 GNN 参数。
